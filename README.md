# Real Estate Sales and Management System
Real Estate Sales and Management System is designed to help Real Estate agents to sell and manage their properties.

**Features**
- Authentication
- Filtering properties based on Price, House, Status, Size, etc.
- View the status of properties, manage users, and manage property information.

**Tech Stack:**
- Java
- ApacheNetBeans IDE

# How to run the project?
- Open the project in ApacheNetBeans IDE.
- Run the `LoginWindow` class to start the application.
